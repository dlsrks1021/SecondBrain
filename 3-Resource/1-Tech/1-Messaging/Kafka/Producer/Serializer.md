
Kafka로 전송할 객체가 단순한 문자열이나 숫자가 아닐 경우  
직렬화 전략을 선택해야 한다.

---

## 직렬화 전략

1. 범용 직렬화 라이브러리 사용
2. 커스텀 직렬화 로직 구현

범용 직렬화 라이브러리 사용이 강력히 권장된다.

---

## Apache Avro

Apache Avro는 언어 중립적인 데이터 직렬화 형식이다.

- JSON 기반 스키마 정의
- 스키마 호환성 지원
- 직렬화 결과에 스키마 ID만 포함

---

## Schema Registry

Avro는 역직렬화 시 스키마 전체가 필요하므로  
Schema Registry 아키텍처 패턴을 사용한다.

- 모든 스키마를 중앙에서 관리
- 레코드에는 스키마 식별자만 포함
- Consumer는 이를 통해 스키마를 조회해 역직렬화한다

이 과정은 Serializer와 Deserializer 내부에서 자동으로 처리된다.
